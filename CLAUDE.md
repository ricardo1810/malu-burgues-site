# Claude Code context - Malu Burger’s Site

## Projeto

Site one-page da Malu Burger’s, hamburgueria local com visual vermelho/amarelo e foco em pedidos via WhatsApp.

## Regras de marca

- Manter energia popular, apetitosa e local.
- Usar vermelho/amarelo como base visual.
- Hambúrgueres, combos e fotos de produto são protagonistas.
- Evitar estética SaaS, tech, IA genérica ou gourmet fria.

## Regras técnicas

- Implementação principal em `site/`.
- Mobile-first.
- HTML semântico.
- CTA WhatsApp claro.
- SEO local básico.
- Rodar build e lint antes de entregar.
