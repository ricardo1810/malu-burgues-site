# Quality gates - Malu Burger’s Site

## Marca

- [ ] O site parece extensão natural do Instagram.
- [ ] Vermelho/amarelo aparecem com intenção, sem poluição visual.
- [ ] Hambúrguer/fotos de produto são protagonistas.
- [ ] Não parece site SaaS, tech ou template genérico.

## Comercial

- [ ] WhatsApp é o CTA principal.
- [ ] Cardápio é fácil de entender no mobile.
- [ ] Combos/destaques aparecem cedo.
- [ ] Textos dão fome e são diretos.

## Técnica

- [ ] Build passa.
- [ ] Lint passa.
- [ ] HTML semântico.
- [ ] Contraste adequado.
- [ ] Responsivo em mobile.
- [ ] Imagens com alt text.
- [ ] SEO básico configurado.

## Anti-IA

- [ ] Sem gradiente roxo/azul genérico.
- [ ] Sem cards 3-colunas iguais como padrão dominante.
- [ ] Sem copy corporativa tipo “eleve sua experiência”.
- [ ] Sem placeholders genéricos finais.
